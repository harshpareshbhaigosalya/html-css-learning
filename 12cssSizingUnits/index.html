CSS Sizing Units Demo

Pixels (px)

width: 200px; height: 50px;

em (relative to element's font-size)

width: 15em; font-size: 20px;

rem (relative to root font-size)

width: 20rem; font-size: 18px;

Percent (%)

width: 60% (of parent)

Viewport Width (vw) & Viewport Height (vh)

width: 40vw; height: 10vh;

auto

width: auto; height: auto;

Mixing Units

width: 50% + 100px; min-width: 200px; max-width: 600px;

em in Nested Contexts

Parent (font-size: 24px)
Child 1 (font-size: 1em = 24px)
Child 2 (font-size: 2em = 48px)

rem in Nested Contexts

Parent (font-size: 32px)
Child 1 (font-size: 1rem = 16px by default)
Child 2 (font-size: 2rem = 32px by default)
